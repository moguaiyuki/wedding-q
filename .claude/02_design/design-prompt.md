# Google Stitch用 デザインプロンプト（3パターン）

## パターン1：格付けチェック風デザイン

### プロンプト
```
Create a luxurious Japanese TV show "Geinojin Kakuzuke Check" inspired quiz app UI design with the following specifications:

Visual Style:
- Primary colors: Gold (#FFD700) and Black (#000000) with deep red (#8B0000) accents
- Secondary colors: Room A = Red (#FF0000), Room B = Blue (#0000FF), Room C = Yellow (#FFFF00) for choices
- Typography: Meiryo font base with thick Ming-style serif for dramatic text
- "映す価値なし" (Utsusu Kachi Nashi) effect: Bold outlined text with drop shadow
- Art Deco inspired golden ornamental frames and decorative borders
- Background: Deep black with subtle marble or velvet texture patterns
- Heavy use of gold foil effects and metallic gradients

Screen Layouts Required:

1. Participant Quiz Screen (Mobile):
- Question category at top (ワイン/音感/味覚 etc.) in gold frame
- Large question text in white on black background
- 2-4 answer buttons in symmetric layout
- Room A (Red background) vs Room B (Blue background) style buttons
- Gold ornamental corners and dividers
- Player status in bottom corner with current rank display

2. Result Screen:
- Door opening animation placeholder
- Large "正解!" in gold or "不正解..." in gray
- Confetti particle effects for correct answers
- Rank change animation: 一流芸能人→普通芸能人→二流→三流→そっくりさん→映す価値なし
- "映す価値なし" shows explosion/smoke effect with bold text overlay
- Explanation text in elegant white box with gold trim

3. Ranking Screen:
- 6-tier rank system visual hierarchy
- Top ranks: Gold ornate frames with crown icons
- Bottom ranks: Hand-drawn cheap nameplate style
- Each entry: Rank badge, nickname, group label (新郎友人/新郎親族/新婦友人/新婦親族), score
- Marble column background with chandelier graphics

4. Presentation Screen (Projector):
- Ultra high contrast black background
- Massive question text with gold Art Deco frame
- Choice boxes with Room A/B/C color coding
- "CHECK 1" "CHECK 2" progression indicators
- Dramatic reveal animations with light rays
- Bar graph statistics in gold gradients

Mobile responsive, Japanese text support, high-end luxury aesthetic throughout.
```

## パターン2：ネプリーグ風デザイン

### プロンプト
```
Create a vibrant Japanese TV show "Nep League" inspired quiz app UI design with full CG aesthetic:

Visual Style:
- Primary colors: Cyan (#00FFFF) to Blue (#0080FF) gradient base
- Team colors: Red team (#FF0000) vs Blue team (#0000FF) for competition
- Background: Dynamic gradients with neon glow effects
- Typography: Geometric sans-serif with custom lettering, 7-segment style for numbers
- High saturation, bright energy like game center atmosphere
- Real-time 3D rendered elements and AR integration feel

Screen Layouts Required:

1. Participant Quiz Screen (Mobile) - "Five League" Style:
- 5 horizontal answer boxes that change from blue to red when correct
- Large question at top with animated speech bubble
- Number badges (①②③④⑤) for each answer position
- Timer bar with color transition (green→yellow→red)
- Score counter with star burst animations
- Glossy 3D buttons with strong shadows

2. Result Screen - "Trokkō Adventure" Style:
- 3D rendered track environment
- Cloud level (70+ correct) → Hill (40-69) → Storm valley (1-39) → Desert (failed)
- Trolley tilting animation based on answer
- Explosion effects for wrong answers
- "○" for correct (green) or "×" for incorrect (red) with particle effects
- Points animation (+100pts) floating upward

3. Ranking Screen - "Percent Balloon" Style:
- 100 balloon graphics that pop based on performance
- Visual progression: Sky → Ground → Desert background
- Colorful alternating row design
- Top 3 on special podium with spotlights
- Each entry with colored rank circle, nickname in fun font, group tag, star icons
- Rainbow gradient effects for top performers

4. Presentation Screen (Projector) - "Hyper Bomber" Style:
- 60-second countdown timer with pulsing effect
- Large bold graphics optimized for TV broadcast
- Question in dramatic banner with shadow effects
- Real-time CG environment changes
- Explosion effects with smoke and debris particles
- Screen shake and flash effects for time up
- Multi-layer video compositing appearance

Full CG rendering, seamless transitions, broadcast-optimized large fonts, Japanese support.
```

## パターン3：平成レトロWeb風デザイン

### プロンプト
```
Create a nostalgic 1990s-2000s Japanese retro website "Heisei Web" style quiz app UI:

Visual Style:
- Pastel palette: Light blue (#ADD8E6), Pink (#FFB6C1), Yellow (#FFFFE0), Mint (#98FB98)
- MS Gothic and MS Mincho style fonts for authentic retro feel
- Pixel art elements with 8-bit aesthetic
- Dotted borders (border: 2px dotted #000)
- Table-based layout appearance with visible cell borders
- Textured backgrounds: Repeating tile patterns (dots, stripes, checker patterns)
- Animated GIF-style decorations (blinking stars ★, rotating items, bouncing arrows)
- Rainbow gradients and WordArt-style headers
- Beveled edges and embossed button effects (border-style: outset)

Screen Layouts Required:

1. Participant Quiz Screen (Mobile):
- Question in HTML table cell with dotted border
- Kawaii pixel art mascot character (^_^) asking questions
- 4 answer buttons with raised 3D bevel effect (different pastel colors)
- Old web counter style score display (like 90s hit counters)
- Blinking "NEW!" badges and animated construction cone GIFs
- Background with seamless tile pattern (16x16 pixel repeat)
- ASCII art decorations ☆･ﾟ･*:.｡..｡.:*･ﾟ☆

2. Result Screen:
- Blinking text effect for result (correct=blue blink, incorrect=red blink)
- Pixel art celebration/consolation character
- Scrolling marquee tag style explanation box
- MIDI sound effect indicators ♪♫🎵
- "Under Construction" animated GIF elements
- ASCII dividers ━━━━━☆☆☆━━━━━
- Frame borders using table cells

3. Ranking Screen:
- Classic HTML table with visible borders (border="1" cellspacing="0")
- Alternating row colors (bgcolor="#FFFFFF" and "#E6E6FA")
- Rank numbers in rainbow gradient text effect
- Nicknames in various web-safe colors
- Group labels as small pixel badges (8x8 icons)
- Pixel star ratings ★★★☆☆
- Animated "UP!↑" "DOWN!↓" arrows in GIF style
- Footer with visitor counter "参加者数: 00060名"

4. Presentation Screen (Projector):
- WordArt-style headers with 3D gradient effects
- Large HTML table with thick borders (border="5")
- Bullet points using special characters (◆◇●○■□)
- Flashing border effect for correct answer (CSS animation)
- Simple bar graphs using table cells with background colors
- ASCII art decorations and emoticons (＼(^o^)／)
- "Best viewed in Internet Explorer 4.0" joke badge
- DHTML-style dynamic effects

Mobile responsive but maintains table structure, inline style attributes, font tags, Japanese text in classic web fonts, maximum nostalgia.
```

## 使用方法

1. 上記のプロンプトをGoogle Stitchにコピー＆ペースト
2. 各パターンで生成されたデザインを確認
3. 気に入ったスタイルをベースに、細部を調整

## 各パターンの特徴まとめ

### パターン1（格付けチェック風）
- **実際の番組要素**: Room A/B/Cの色分け、6段階ランクシステム、「映す価値なし」演出
- **カラー**: 黒・金・赤/青の対比
- **雰囲気**: 高級感、緊張感、ドラマチック
- **適している場面**: フォーマルな披露宴、大人っぽい演出

### パターン2（ネプリーグ風）
- **実際の番組要素**: ファイブリーグ、トロッコアドベンチャー、パーセントバルーンなどのコーナー別UI
- **カラー**: シアン/青のグラデーション、赤青チーム対戦色
- **雰囲気**: フルCG、ゲームセンター的、エネルギッシュ
- **適している場面**: カジュアルな披露宴、盛り上げたい演出

### パターン3（平成レトロWeb風）
- **要素**: HTMLテーブル、ドット絵、点滅テキスト、カウンター
- **カラー**: パステルカラー、虹色グラデーション
- **雰囲気**: ノスタルジック、親しみやすい、ユーモア
- **適している場面**: 友人中心の披露宴、笑いを取りたい演出
